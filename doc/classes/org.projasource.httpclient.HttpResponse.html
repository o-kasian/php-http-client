__construct()

__construct(resource $resource)

Creates new HttpResponse, reading it from input sream of the given resource.

In most cases Response is not constructed manually, however it may be 'mocked' by opening a file containing raw http response, and passing file descriptor to a constructor f.ex. 

$fp = fopen('response.raw');
$resp = new HttpResponse($fp);

getHeaders()

getHeaders() : array

Returns http headers in a form of array.

NOTICE! All header names will be lowercased, as far as they have to be case insensitive. If you need to obtain an information about a concrete header, use {@see getHeader}

Returns

associative array $header_name => $header_value

getHeader(string $name) : string

getRawBody()

getRawBody() : string

Returns a 'raw' http entity.

No conversion is made, and everything after header section of the response (without leading CRLF) is returned.

Returns

string —

response body
