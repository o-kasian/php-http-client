      Starting point for making a http requests.

      Uses builder pattern making your code easy to write, and easy to understand. A basic example of usage would be:
      
$response = HttpRequest::build("https://example.com")
    ->method("POST")
    ->contentType("application/json")
    ->entity(json_encode($entity))
    ->proxy("http://user:password@192.168.0.1:8080")
    ->accept("application/json")
    ->execute();

echo $response->getEntity()->field;

      A content type of the entity.

      contentType(string $type) : \org\projasource\httpclient\HttpRequest
      Asumes a format of [type "/" subtype], as described in RFC 2615 section 3.6.1. Content type is further used to convert entity into string (if necessary) while sending a request body, and is set into 'Content-Type' header. NOTICE! Any data after semicolon (";") will be lost.
