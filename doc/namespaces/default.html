httpClientAutoLoad()

httpClientAutoLoad(\type $className) : boolean

HttpClient autoloader.

If you do not use Composer, you can add this in the begining of your __autoload chain f.ex. 


function __autoload($className) {
  if (httpClientAutoLoad($className)) {
    return;
  }
  //your code here
}

Parameters

\type $className

name of the class to be looked for

Returns

boolean —

if the following class was loaded
